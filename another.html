Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM. It's given below: 1. getElementById, 2. getElementsByClassName, 3. querySelectorAll, 4. querySelector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The main difference among them is how they handle html tags.innerText sets the visible text. However textContent sets visible,invisilbe text both and innerHTML sets whole text,tags,styles etc.

Question-3: What is event delegation in the DOM?

Event delegation is a process in JavaScript by whom a single event listener is added to a parent element to handle events for child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM event propagation model where an event starts at the target element and "bubbles up" to the parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element : document.createElement(). To add an element: parentElement.appendChild(). To remove an element: parentElement.removeChild().
